# Slidev Deck for AI Agent Guardrails

AI エージェント実行時のコマンド確認を減らすために行った、
ガードレール整備と sandbox 有効化の取り組みをまとめた Slidev スライドです。

## Demo

https://github.com/user-attachments/assets/f0be0ff8-bad9-4351-a215-9f68d59e8809

## Run Locally

```bash
npm install
npm run dev
```

Open <http://localhost:3030/>

## Files

- [`slides.md`](slides.md): Slidev のスライド本体
- [`package.json`](package.json): ローカル起動用の npm scripts

## Development

スライド本文は [`slides.md`](slides.md) を編集します。

Slidev のクリック表示は `<v-click>` / `<v-clicks>` で制御しています。
